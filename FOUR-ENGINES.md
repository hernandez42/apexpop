# 四引擎自进化系统 — Zircon 熵世界核心

## 核心理念
自进化不是"自己改自己"，是"自己出题→自己验证→自己筛选→自己保留"的闭环。
C core + 配套 LLM = 真核心。最优选择 → 交叉基因 → 基因突破。

## 四个引擎

### 引擎 1：自出题（Self-Propose）
- 不等别人问，自己生成难题
- 难度感知：太简单/太难都不要，只要"刚好能学会"的
- 机制：可学习性奖励 LR = 4p(1-p)
- 来源：Absolute Zero Reasoner

### 引擎 2：自验证（Self-Verify）
- 不是自评自欺，是用外部标准检验
- 机制：代码执行器验证技术任务、用户反馈验证服务任务、benchmark 验证能力
- 来源：R-Zero 不确定性奖励

### 引擎 3：自筛选（Self-Filter）
- 不是什么都存，只留高信息量的经验
- 机制：信息量评估 ICS，< 0.3 丢弃
- 来源：EvoAgentX 评估层

### 引擎 4：防退化（Anti-Degradation）
- 不是无限向上，有节奏地进化
- 机制：固定 benchmark 检测、退化自动暂停、外部锚点校准
- 来源：R-Zero 退化警告
