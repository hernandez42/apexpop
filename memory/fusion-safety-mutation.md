### 2. 安全审计层实现

#### 安全审计函数 `safety_audit(mutation_candidate)`
返回 `{"pass": bool, "reason": str, "checks": dict}`

**7 项安全检查**：

| # | 检查项 | 说明 | 失败后果 |
|---|--------|------|----------|
| 1 | `valid_dimension` | 维度名称在合法列表中 | 直接拦截 |
| 2 | `value_in_range` | 新值在安全范围内 [min, max] | 直接拦截 |
| 3 | `direction_constraint` | 安全维度(T_c, S_v)只升不降 | 直接拦截 |
| 4 | `step_size_ok` | 步长不超过安全上限 | 直接拦截 |
| 5 | `no_critical_drop` | 无临界跌落（阈值 0.3） | 直接拦截 |
| 6 | `no_dangerous_ops` | 变异类型不含危险操作模式 | 直接拦截 |
| 7 | `toward_target` | 变异朝向目标方向 | 直接拦截 |

#### 安全边界规则定义 (`SECURITY_RULES`)
```python
"blocked_dimensions": []  # 动态判断
"protected_files": ["~/.openclaw/openclaw.json"]
"value_constraints": {
    "T_c": {"min": 0.0, "max": 2.0, "direction": "up_only"},
    "S_v": {"min": 0.0, "max": 2.0, "direction": "up_only"},
    "C_m": {"min": 0.5, "max": 2.0},
    "D_s": {"min": 0.5, "max": 2.0},
    "O_e": {"min": 0.5, "max": 2.0},
    "E_co": {"min": 0.5, "max": 2.0},
    "P_i": {"min": 0.5, "max": 2.0},
}
"max_step_sizes": {"T_c": 0.3, "S_v": 0.3}
```

### 3. 管道集成

变异引擎 v3.0 流水线：
```
诊断 → 变异生成 → 🔒安全审计(最高优先级) → 常规验证 → 三重过滤 → 经验沉淀
```

**关键设计**: 安全审计在所有其他检查之前执行，违反安全边界的变异候选直接丢弃，不进入评估。

## 四、核心洞察

### 进化搜索空间 = 能力空间 ∩ 安全约束

```
能力空间（所有可能的变异）:
  ┌──────────────────────────────┐
  │  D_s  O_e  C_m  T_c  S_v   │
  │  E_co  P_i                   │
  │  ...                         │
  └──────────────────────────────┘

安全约束（硬边界）:
  ┌──────────────────────────────┐
  │  T_c, S_v 只升不降           │
  │  所有维度值 ∈ [0.5, 2.0]    │
  │  步长 ≤ 0.3 (安全维度)       │
  │  无临界跌落                  │
  │  无危险操作                  │
  │  朝向目标方向                │
  └──────────────────────────────┘

进化搜索空间 = 能力空间 ∩ 安全约束
  ┌──────────────────────────────┐
  │  ✅ 只有通过安全审计的变异   │
  │     才能进入评估流程         │
  └──────────────────────────────┘
```

### 安全基因与变异基因的融合关系

| 基因类型 | 功能 | 在管道中的位置 |
|----------|------|---------------|
| 变异基因 | 生成变异候选 | 第二层：变异层 |
| 安全基因 | 审计变异候选 | 🔒 第零层：安全审计（最高优先级） |
| 信息基因 | 过滤低信息量 | 第三层：三重过滤 |
| 学习基因 | 加权中等难度 | 第三层：三重过滤 |

**核心原理**: 安全基因不是变异的"惩罚项"，而是变异的"边界条件"。变异在边界内自由探索，但绝不能突破边界。
