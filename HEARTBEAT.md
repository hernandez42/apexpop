# HEARTBEAT.md

## 市场监控任务

每次心跳执行以下检查：

1. **行情异动检查**
   - 用 market-radar 拉取指数数据
   - 检查是否有指数涨跌幅 > 2% 的异常波动
   - 如有异动 → 立即推送群聊

2. **热门个股监控**
   - 检查自选股是否有涨停/跌停
   - 发现异动 → 推送提醒

3. **资讯速递**
   - 搜索最新财经要闻
   - 重大政策/事件 → 推送摘要

4. **技能组合验证**
   - 检查 SKILL_ORCHESTRATION.md 中的模板是否正常执行
   - 发现问题 → 自动修复

## 执行频率
- 市场交易时段（9:30-15:00）：每次心跳都检查
- 非交易时段：跳过市场监控

## 🧬 自进化循环（新增）

每次心跳触发自进化循环（四引擎联动）：
```bash
python3 glue.py --evolve
```

每次心跳触发回音壁循环（知识循环增强）：
```bash
python3 scripts/echo-wall.py
```
循环内容：
1. 代入璇玑+APEX公式 → 计算当前状态
2. 识别短板（< 1.0 的基因）
3. 天兵天将解决（搜索→提取→应用）
4. 反问自己（还有问题吗？）
5. 记录到 EVOLUTION_LOG.md
6. 循环继续

## 🧠 思维模式（SOUL 融合）

每次心跳检查：
- 是否主动发现了问题（不等别人问）
- 是否解决了真实问题（不是算公式）
- 是否反问了自己（还有提升空间吗）
- 是否产生了新洞察（跨域融合）

如果以上全部为"否" → 提醒自己要主动思考
