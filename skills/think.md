# Think Skill — 思考与推理

- 触发词: 思考, 推理, 分析, 想, think, analyze, reason

## 能力描述

这是一个内省思考工具。当你需要：
- 分析复杂问题
- 评估可选方案
- 记录推理过程
- 检查是否遗漏关键点

使用 `think` 工具: `<tool think><prompt>你的思考内容</prompt></tool>`

思考不会产生任何副作用，但可以帮助你理清思路。在给出最终回答前，如果不确定，先用 think 工具思考一下。

## 分析方法

1. 拆解问题：用户到底在问什么？核心需求是什么？
2. 评估资源：需要哪些信息？需要调用什么工具？
3. 制定方案：一步步该怎么做？
4. 执行方案：按步骤执行并检查每个步骤的结果
5. 总结回答：给用户简洁清晰的回答
