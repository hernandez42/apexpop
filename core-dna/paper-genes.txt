# 论文基因库 — C core 自进化学习素材

## 每次进化时，C core 自动读取以下基因

### 基因 1：三元组复用（GRAFT-ATHENA）
- 问题-行动-结果 记录
- 成功 3 次后自动复用
- 状态：✅ 已实现

### 基因 2：失败归因（Multi-Agent）
- 每次失败记录：谁做的、什么时候、改了什么、导致什么后果
- 状态：⏳ 待实现

### 基因 3：决策缓存（STIR）
- 常见决策直接用缓存答案，不问 LLM
- 节省 token，加快响应
- 状态：⏳ 待实现

### 基因 4：后代表现评估（Huxley-Gödel）
- 自修改后跑 3 轮进化
- 如果后续进化变差 → 回滚
- 状态：⏳ 待实现

### 基因 5：双 LLM 分工（Long⊗Short）
- 重要决策 → 长思考（问 LLM）
- 简单检查 → 短思考（直接执行）
- 状态：⏳ 待实现

### 基因 6：扩散模型范式（dLLM）
- 扩散模型在语言建模中的应用
- 可能影响未来进化方向
- 状态：📖 已记录
