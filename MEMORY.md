### Context Engineering（核心发现）
- KV-cache 命中率是 Agent 最关键运维指标（cached 比 uncached 便宜 10 倍）
- 文件系统 = 终极上下文：无限容量、持久化、可操作
- 保留错误信息：清理错误 = 丢弃训练数据
- "复述"操控注意力：关键信息放在上下文末尾

### Meta-Harness（核心发现）
- 摘要不是压缩，是丢失因果信息
- 完整执行轨迹 > 分数+摘要
- 自动发现的策略超越人类设计
- Agent 优化 Agent 的评估基础设施

### PreFlect（核心发现）
- 从事后复盘到事前挑刺
- Pre-mortem：在执行前预想失败场景
